# 🎙️ VoiceWave - دليل التشغيل الكامل

## هيكل الملفات
```
voicechat/
├── server.js          ← الباك اند (Node.js)
├── package.json       ← المكتبات
└── public/
    └── index.html     ← الواجهة الكاملة
```

---

## 🚀 الطريقة 1: Railway (مجاني + أونلاين للعالم) ✅ موصى بها

### الخطوات:

**1. أنشئ حساب على Railway**
- اذهب لـ https://railway.app
- سجل بـ GitHub

**2. ارفع الكود على GitHub**
- اذهب لـ https://github.com/new
- أنشئ مستودع جديد اسمه `voicechat`
- ارفع الملفات الثلاثة (server.js, package.json, public/index.html)

**3. على Railway:**
- اضغط "New Project"
- اختر "Deploy from GitHub repo"
- اختر مستودع `voicechat`
- سيشتغل تلقائياً!

**4. احصل على الرابط:**
- Settings → Networking → Generate Domain
- ستحصل على رابط مثل: `voicechat-production.up.railway.app`

**هذا الرابط يشتغل من أي دولة في العالم ✅**

---

## 🚀 الطريقة 2: Replit (أسهل للمبتدئين)

**1.** اذهب لـ https://replit.com
**2.** أنشئ Repl جديد → اختر Node.js
**3.** انسخ محتوى server.js في الملف الرئيسي
**4.** أنشئ مجلد `public` وضع index.html داخله
**5.** انسخ محتوى package.json
**6.** اضغط Run ✅

---

## 🖥️ الطريقة 3: جهازك المحلي (للتجربة فقط)

```bash
# 1. ثبت Node.js من nodejs.org

# 2. افتح Terminal في مجلد المشروع
cd voicechat

# 3. ثبت المكتبات
npm install

# 4. شغل السيرفر
npm start

# 5. افتح المتصفح
# http://localhost:3000
```

⚠️ **المشكلة:** لو شغلته محلياً، الناس الثانية ما يقدرون يوصلون إليك.
✅ **الحل:** استخدم ngrok للتشغيل المؤقت:
```bash
# بعد تشغيل السيرفر، في terminal ثاني:
npx ngrok http 3000
# سيعطيك رابط مؤقت يشتغل من أي مكان
```

## 🎯 كيف يشتغل التطبيق

1. **افتح الرابط** في المتصفح
2. **أدخل اسمك** وكود الغرفة (أي كلمة أو رقم)
3. اضغط **"انضم"** أو **"إنشاء غرفة جديدة"**
4. **شارك كود الغرفة** مع صديقك
5. صديقك يفتح نفس الرابط ويدخل نفس الكود ← يبدأ الاتصال! 🎉

## 🔧 أسئلة شائعة

**Q: الصوت ما يوصل؟**
A: تأكد من إعطاء إذن الميكروفون للمتصفح. اضغط على أيقونة القفل في شريط العنوان.

**Q: يشتغل مع كم شخص؟**
A: يشتغل بشكل مريح مع 2-8 أشخاص. لأكثر من ذلك يحتاج تحسينات إضافية.

**Q: هل البيانات مشفرة؟**
A: نعم، WebRTC يشفر الصوت تلقائياً (DTLS/SRTP).

**Q: كيف أضيف ميزات أكثر؟**
A: راسلني في كلود وأضيف أي ميزة تبيها! 😊
